# Global template install

Copy these files into your personal tool configuration:

- `codex-home-AGENTS.md` -> `~/.codex/AGENTS.md`
- `claude-home-CLAUDE.md` -> `~/.claude/CLAUDE.md`

These files are intentionally generic.
They should define your cross-project working style, not repository-specific commands.
Use each repository's own `AGENTS.md` or `CLAUDE.md` for project details.
